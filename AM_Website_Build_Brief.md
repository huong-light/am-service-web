# AM Website Build Brief

## Brand

- Brand name: **AM Service & Technology**
- Domain: **amservice.vn**
- Email: **admin@amservice.vn**
- Style: very dark red background, white text, modern, technical, clean, serious.
- Avoid colorful neon, blue/purple AI gradients, or playful startup style.

## Web Sections

Build a simple professional website with:

1. **Hero**  
   Introduce AM Service & Technology as a technical solution provider for small and medium venues.

2. **Services**  
   Core services:
   - MA3 onPC System
   - Resolume Mapping
   - AI Agent Automation
   - Workflow Training
   - Vectorworks / Capture / Depence Workflow

3. **Process**  
   Show the workflow:
   - Consult
   - Design
   - Build system
   - Test
   - Train user
   - Support

4. **Contact**  
   Show clearly:
   - Website: amservice.vn
   - Email: admin@amservice.vn

5. **Footer**  
   Keep it simple, dark, and clean.

## Code Rules

When working with GitHub files:

- If the file is new, use **Create**.
- If the file already exists, use **Read** first to get the file `sha`, then use **Update**.
- Always write the **full file content**.
- Do not send patch fragments.
- Do not partially edit files.
- Keep code clean, readable, and production-ready.
